# css-utils

CSS library with utility-classes

## Development

### Prerequisites

- Node.js and npm

### Setup

1. Clone the repository

```bash
git clone https://github.com/ivteplo/css-utils
```

2. Navigate to the folder

```bash
cd css-utils
```

3. Install dependencies

```bash
npm install
```

4. Happy hacking! ðŸŽ‰

### Scripts / tasks

- Make a new CSS build

```bash
npm run build
```

- Rebuild on changes

```bash
npm run watch
```

- Format source files

```bash
npm run format
```
